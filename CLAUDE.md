# Project constraints

Stack:
- React + Vite
- Node.js + Express
- MongoDB + Mongoose

Priorities:
1. Working functionality
2. Deployment reliability
3. Simplicity
4. Clean code

Rules:
- Do not add dependencies without explaining why.
- Do not introduce authentication unless requested.
- Do not create unnecessary abstractions.
- Keep controllers and routes simple.
- After each implementation step, run the relevant tests or build.
- Never modify deployment configuration without explaining the change.
- Prefer the smallest working solution.
